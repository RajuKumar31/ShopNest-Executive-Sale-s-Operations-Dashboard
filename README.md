# ShopNest Executive Sales & Operations Dashboard (Power BI)
**Project Overview**

This project presents an interactive **Executive Sales & Operations Dashboard** built using **Power BI**.

The dashboard analyzes multi-table e-commerce data to evaluate:

Sales performance
Operational efficiency
Customer behavior
Product performance
Delivery performance

The objective was to provide executive-level insights for strategic decision-making.
